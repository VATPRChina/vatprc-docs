# 广州白云国际机场（ZGGG）标准运行程序

## 1. 总则

### 1.1 根据

为了确保 VATPRC 管辖范围内的广州白云机场（ZGGG）安全、合规和高效的运行，依据中国民用航空局（CAAC）发布的[《中华人民共和国航空资料汇编（AIP CHINA）》](https://www.eaipchina.cn)，结合 VATPRC 实际运行情况，制定了适用于广州白云机场（ZGGG）的标准运行程序。

### 1.2 原则

管制员上线时需遵守 [VATSIM Code of Conduct (COC)](https://vatsim.net/docs/policy/code-of-conduct)，并积极与上下级管制员协调以达成一致认知。需要实施与本文件不一致的运行方式时，需与相关管制员协调达成一致后实施。

### 1.3 适用范围

本文件适用范围是 VATPRC 管辖内广州白云机场的塔台、地面、放行、机坪席位。

> [!CAUTION]
> 本文件严禁用于真实运行!

- 本文件中出现的“离场点”一词指：离场程序之后巡航航段的第一个定位点/导航台/航路点。

#### 2.5 标准滑行路线

管制员可指挥航空器在标准滑行路线的中间节点或终点等待或加入其他滑行道继续滑行。当管制员在标准滑行路线的中间节点指挥航空器加入其他滑行道时，意味着取消已发布标准滑行路线的剩余部分。

## 4. 席位职责

### 4.1 塔台

#### 4.1.1 离场航空器的放飞规则和移交

> [!NOTE]
> 下列标注的“间隔”如非特别注明，则一律以机组离地（有正上升趋势）为计时起点。

离场航空器放飞需要满足下列条件：

1. 相同跑道满足其尾流间隔
2. 对于不同跑道同一离场点航空器采取最少 2 分钟（120 秒）的间隔
3. 对于同组平行近距跑道，进近航空器在距离跑道末端 10 公里内时，离场航空器应于进近航空器接地且有减速趋势后放飞
4. 对于同组平行近距跑道，进近航空器复飞后，在复飞航空器通过跑道头 2 分钟（120 秒）后放飞
5. 对于同组平行近距跑道或相同跑道，航空器执行起落航线时，离场航空器在起落航线航空器一转弯后放飞
6. 跑道上没有其他航空器

#### 4.1.2 复飞航空器的处理

仪表进近航空器根据标准复飞程序执行，在航空器有正上升趋势后移交给进近。目视飞行规则航空器复飞后加入本场起落航线。

#### 4.1.3 起落航线

A、B 类航空器高度 300 m，C、D 类航空器高度 500-600 m。

#### 4.1.4 目视飞行离场航空器的处理

除经过广州进近或白云塔台允许，在广州进近和塔台管制区内飞行，必须按照仪表飞行规则飞行。
